# DevTask

A modern MERN Stack Task Management Application that helps users organize, manage, and track daily tasks with a clean, responsive, and user-friendly interface.

---

## рҹ“Ң About

DevTask is a full-stack Task Management application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js).

The application allows users to securely register, log in, manage their personal tasks, update their profile, and monitor productivity through an interactive dashboard. It is designed with a modern UI, responsive layout, and secure authentication using JWT.
